🧩 Балда – Игра в слова на 7 языках
Классическая игра «Балда» в терминале!
Составляйте слова из букв на поле 5×5, соревнуйтесь с другом или компьютером.
Развивайте словарный запас, тренируйте мышление и получайте удовольствие!
Реализована на 7 языках программирования с единым интерфейсом и цветным выводом.

🚀 Возможности
Игровое поле – квадрат 5×5, заполненный случайными буквами (гласные и согласные).

Правила – слова составляются из соседних (по горизонтали, вертикали или диагонали) клеток, каждая клетка используется не более одного раза.

Два режима:

vs – игра против другого игрока (по очереди).

ai – игра против компьютера (простого ИИ).

Подсчёт очков – количество букв в слове (длина слова = очки).

Словарь – встроенный набор русских слов (можно легко расширить).

Проверка слова – слово должно существовать в словаре и быть составлено по правилам.

Интерактивный ввод – игрок вводит координаты клеток в порядке обхода.

Цветной вывод – буквы, подсветка текущего хода, сообщения.

Статистика – отображение текущего счёта, ходов, оставшихся возможных слов.

Автосохранение – состояние игры сохраняется (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <режим> [опции]
Режимы
vs – два игрока.

ai – игра против компьютера.

Опции
-d, --dict <file> – путь к файлу словаря (по умолчанию используется встроенный).

-s, --save – сохранять состояние игры в файл.

-h, --help – справка.

Управление
Ввод координат клеток в формате A1 B2 C3 ... (буква столбца A–E, цифра строки 1–5).

q – выход.

? – показать подсказку (доступные ходы) – в некоторых реализациях.

Примеры
bash
# Игра против компьютера
python balda.py ai

# Игра с другом
python balda.py vs

# Использовать внешний словарь
python balda.py ai -d my_words.txt

🧠 Логика игры
Поле – 5×5 клеток с буквами (генерируется случайно, но гарантируется наличие гласных).

Ход игрока – игрок вводит последовательность клеток (например, A1 B1 C1), программа проверяет:

Все клетки соседние (по 8 направлениям).

Каждая клетка используется только один раз.

Полученное слово есть в словаре.

Слово не было использовано ранее в этой партии.

Подсчёт очков – длина слова.

ИИ – ищет все возможные слова на поле, выбирает самое длинное (или случайное из длинных).

Окончание игры – когда игрок (или оба) не могут найти ни одного нового слова, игра заканчивается.

✨ Дополнительные фичи
Встроенный словарь – содержит 200+ русских слов длиной от 3 до 7 букв.

Подсветка – текущее слово выделяется на поле.

Подсказка – ИИ может подсказать одно из возможных слов (опционально).

Сохранение – состояние игры сохраняется в ~/.balda_save.json.
